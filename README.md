# 🚀 Complaint Tracking System (Spring Boot + MongoDB)

## 📌 Overview
This is a backend project built using **Spring Boot** and **MongoDB**.

It allows users to:
- Raise complaints
- Track complaint status
- Admin can manage and update complaints

---

## 🧰 Tech Stack

- Java 17
- Spring Boot
- Spring Web
- Spring Data MongoDB
- Lombok
- Maven
